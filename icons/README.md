# PWA Icons

Place your app icons in this folder with the following specifications:

## Required Icons

- **icon-192.png** - 192x192 pixels
- **icon-512.png** - 512x512 pixels

## How to Create Icons

You can use the scale emoji ⚖️ as your icon:

1. Go to a favicon generator like:
   - https://favicon.io/favicon-generator/
   - https://realfavicongenerator.net/

2. Or create simple icons with the ⚖️ emoji:
   - Open any design tool (Figma, Canva, etc.)
   - Create a 512x512 square with dark background (#0a0a0f)
   - Place the ⚖️ emoji in the center
   - Export as PNG
   - Resize to 192x192 for the smaller icon

3. Save the files here as:
   - `icon-192.png`
   - `icon-512.png`

## Current Status

Once you add the icons, your app will be installable as a Progressive Web App (PWA) on mobile devices and desktop browsers.
